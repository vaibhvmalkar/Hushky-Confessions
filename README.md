# Hushly

A brief description of what this project does and who it's for

### What This Project Does

This is a web-based anonymous confession platform that allows users to submit confessions without revealing their identity. The system enables people to share their thoughts, secrets, or feelings anonymously through a simple web form, with all submissions stored securely in Firebase Realtime Database.

**Core Features:**
- Anonymous submission system with Firebase Anonymous Authentication
- Beautiful sending animation with paper plane effect when confessions are submitted
- Admin dashboard to view all received confessions
- Share confessions to Instagram Stories functionality
- User reporting and permanent ban system for inappropriate content
- Real-time database synchronization
- Mobile-responsive design with animated backgrounds and confetti effects

### Who It's For

**Primary Users:**
- **Social Media Influencers & Content Creators**: Instagram users who want to engage their followers by collecting anonymous confessions, questions, or feedback to share on their stories
- **Students**: College and university students creating confession pages for their campus communities
- **Online Communities**: Group admins who want to gather honest feedback or stories from members anonymously

**Use Cases:**
- Instagram story engagement
- Campus confession pages
- Anonymous feedback collection
- Mental health support communities where anonymity is crucial
- Secret sharing platforms

**Technical Profile:**
This project is ideal for developers and students learning full-stack web development, Firebase integration, and modern CSS animations. It demonstrates practical implementation of authentication, database management, security rules, and responsive design - making it perfect for your cybersecurity and web development learning goals as a Computer Science student.

The platform prioritizes user privacy through anonymous authentication while maintaining admin control through the ban system and Firebase Security Rules.

## Run Locally

Clone the project

```bash
  git clone https://github.com/vaibhvmalkar/hushly.git
```

Go to the project directory

```bash
  cd hushly
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
